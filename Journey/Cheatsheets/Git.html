GIT
  • git clone "git@github.com:USER-NAME/REPOSITORY-NAME.git": (To connect the github repository to the local git)

  • git add "filename": (to add a new/modified file from working directory to the staging area)

  • git rm: (to remove a file from git)

  • git commit -m "information about process": (to commit the staged files)

  • git status: (to check where the changes are as per git{staged/working etc.})
    1. Red Color: The file has not been staged yet
    2. Green Color: The file has been staged
    3. Nothing to commit,working tree clean: The file has been comitted into git

  • git push: (to push the commited changes to github)

  • git log: (to check the log history of changes or snapshots)
