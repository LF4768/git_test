FLEXBOX

How To Start Flexbox

    Choose/Make a container to Flex like div, span, p, etc
  • Display: This is used to change the display of your page from normal flow
    The flex style changes your container to a flex container and the items in it are called flex-items

Axes

    Flexbox aligns items on the basis of two axes that are perpendiculer to each other:
  • Main Axis: Alignments occur parallel to this axis
  • Cross Axis: Alignments occur perpendiculer to this axis

    • By default, the Main axis goes from right to left while the Cross axis goes from top to bottom which can be changed by using:

  • flex-direction: Used to change the direction of the flex axes
    There are 4 different directions
  1. Flex-direction:row
    This Is Row

Flex Shorthand

    Flex has 3 more properties namely:
  • flex-grow: Specifes how much area the container takes with respect to the webpage and other containers
    Negative values are invalid
    I Dont Have grow value
    I Have grow value
    I Dont Have grow value
  • flex-shrink: specifes the rate at which area of the container shrinks with respect to the webpage and other containers
    Negative values are invalid
    I Dont Have shrink value
    I Have shrink value
  • flex-basis: Sets the size of the items with respect to the container
    There are tricks to specify their values easily:(all are respectively)
  • auto: sets the values as 1 1 auto
  • initial: sets the values 0 1 auto
  • none: sets the values 0 0 auto
