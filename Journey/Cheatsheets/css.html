CSS


Adding CSS to HTML

    Three ways to add CSS:
    1. External CSS:
      using "link" tag with attribute rel="stylesheet" and href="./file location"
    2. Internal CSS:
      using style element in the head element
    3. Inline CSS:
      using style attribute in the opening tag of an element


Different Types of Selectors:

  • *(Universal Selector): Applies the style properties to all the elements

  • Type Selectors: used to select all the tags of one type like div, p

  • Class Selectors: used to select the elements with the class attribute
    syntax for class attribute is .'name of tag' like .div , .p
    two or more classes can be made with a space between them in the double quotes

  • ID Selectors" used to select the tag with the id attribute
    syntax for id attribute is #'name of tag' like #div, #p
    many tags of the same or different type can have the class attribute but an id attribute is unique to only 1 tag(not even element of same type)

  • Ways to apply same styles to multiple tags:
    1. Grouping Selectors: using comma(,) to group 2 attributes together like
      .div,
      .p {}

    2. Chaining Selectors using style syntaxes together to chain the tags like
      .div.p(no space) {}

    3. Descendant Combinator: used to style the specific tags that follow the ancestory i.e. that come under all the attributes defined with syntax
      syntax is .'name of tag" (space) .'name of tag' like .div .p

Properties

*use colon(:) after property and semicolor(;) after value*
  • Color: changes the text color

  • background-color: changes the background color of the tag(tag only)

  • font-family: used to provide font family name("Times New Roman") or/and generic family name(sans-serif) to the browser
    Comma is used to provide more than 1 fonts

  • font-size: used to change the size of the text
    syntax is 'number'px (no space) like 20px, 35px

  • font-weight: changes the boldness of the text
    700 is the equivalent of bold

  • text-align: changes the position of the text horizontally

  • height and width(image): changes the height and width of the image
    Use auto to keep the proportions same as the original image
