BASH


  • cd: to change directory
    use {cd . .} to go back one directory and slashes to keep going
    {. . goes back one from the pwd and not any other specific directory}

    Use {cd -} to go back to the directory you were in before

  • pwd: print working directory i.e. to check the directory position from root

  • ls: to show contents of current directory

  • "command" --help: to check the possible arguments and their working

  • mkdir: to create a directory

  • echo "text": to print text in the terminal
    use echo > to put text in a file or create an empty file can also use touch but only for empty file
    To add/change text in a given file:
    1. > overwrites the text if any
    2. >> adds the text as a new line if two texts written, it adds both in the same line

  • rm: to remove a file To remove a directory use -r argument, also use -i for prompt before deleting
    rmdir is BETTER to use.

  • mv "directory if needed/filename" "new location/filename": to move a file or directory from one location to another
    To change the name of a file only: "directory if needed/filename" "new name"
    CAUTION: if files have same name mv overwrites the file without prompt

  • cat/less: to check the contents of the file
    -w for word count, -l for lines and -m for characters

  • clear: to clear the bash shell

  • wc: to count the number of words in a file

  • head/tail -n "number": To take specific number of lines from the top{by head} or the bottom{by tail}
    without -n, it will take all the lines

  • | : to perform multiple actions in one step

  • sort:to sort the items on a list
    -r for reverse order
